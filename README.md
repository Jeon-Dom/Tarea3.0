# Tarea 3.0 - Automatización de CI/CD con GitHub Actions y Docker

Este repositorio contiene una aplicación básica de calculadora programada en Python, automatizada mediante un flujo de CI/CD con GitHub Actions y empaquetada utilizando Docker.

## 🚀 Características del Proyecto
- **Aplicación:** Calculadora Simple (`ejercicio:3.0.0`).
- **Pruebas:** Automatizadas utilizando `pytest`.
- **Contenedor:** Dockerfile optimizado.
- **CI/CD:** Pipeline que prueba, ejecuta, simula despliegue y publica la imagen en GitHub Container Registry (GHCR).
